# Linear Programming Simplex Solver
## Requirements
Python 3.7+
pip

## How to Run
### Use Precompiled Preview
```cmd
git clone https://github.com/Pastoray/Simplex-Solver
cd Simplex-Solver/preview
./app.exe
```

### Or Build Yourself
```cmd
pip install -r requirements.txt
python app.py
```
after these commands succeed, open your browser to `http://localhost:5000`
